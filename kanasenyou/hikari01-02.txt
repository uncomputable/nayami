『 借[か]り物[もの] 』

日本[にっぽん] の 人[ひと] は 今[いま] 重[おも]い 重[おも]い 荷物[にもつ] を 背負[せお]って 世界[せかい] の 人々[ひとびと] と 競争[きょうそう] を 続[つづ]けて いる。 汗[あせ]みどろ に なって いくら 走[はし]って も その 荷物[にもつ] が 邪魔[じゃま] に なって 思[おも]う よう に 走[はし]れない。

その 荷物[にもつ] を 下[お]ろせば よい の に 日本[にっぽん] の 人[ひと] は その 荷物[にもつ] を 宝[たから] の よう に 大事[だいじ] に 思[おも]って なかなか 捨[す]てよう と は しない。

重[おも]い 荷物[にもつ] と いえば 何[なに] か？

それ は 支那[しな] から の 借[か]り物[もの]！

お-互[たが]い に 気[き] が つけば、 返[かえ]して しまおう では ない か。
