『 改良[かいりょう] 仮名文字[カナモジ] を まず 児童[じどう] に 植[う]えつけて 』

我[わ]が 国字[こくじ] の 改良[かいりょう] が できる か、 どう か。 議論[ぎろん] して いる だけ で は、 いつ に なって も 解決[かいけつ] が つきませぬ。 幾千遍[いくせん-べん]、 幾万遍[いくまん-べん] と 読[よ]み慣[な]れた 漢字交[かんじ-ま]じり の 文[ぶん] と、 ただ 一回[いっ-かい] また は 数回[すう-かい] くらい しか 目[め] に 触[ふ]れた こと の ない この 字体[じたい] と を 比較[ひかく] して、 その 是非優劣[ぜひ-ゆうれつ] を 判決[はんけつ] される こと は 苛酷[かこく] です。

議論[ぎろん] より も、 むしろ これら の 改良案[かいりょう-あん] を 実地[じっち] に 植[う]えつけて みて、 これ が 国民[こくみん] の 間[あいだ] に 生育[せいいく] する か いな か を 試[ため]す が 問題解決[もんだい-かいけつ] の 近道[ちかみち] です。 本協会[ほん-きょうかい] は 「アヒリ ノ シリフリ」 と 題[だい]した 童謡[どうよう] の 絵本[えほん] を 発行[はっこう] し、 この 絵本[えほん] に 親[した]しませ ながら、 この 改良案[かいりょう-あん] の 苗[なえ] を 児童[じどう] に 植[う]えつけて みよう と 企[くわだ]てた の で あります。
