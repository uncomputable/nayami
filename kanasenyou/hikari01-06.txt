『 矢立[やたて] と タイプライター 』

ある 人[ひと] から こんな 話[はなし] を 聞[き]いた。 それ は 汽車中[きしゃ-ちゅう] で 目撃[もくげき] した と いう 実話[じつわ] で ある。

乗[の]り合[あ]い の 旅客[りょかく] の 一人[ひとり] が、 腰[こし] なる 矢立[やたて] を 取[と]り出[だ]して 何事[なにごと] か 書[か]き始[はじ]めた。 幾度[いくど] も 筆[ふで] を なめ ながら 記者[きしゃ] の 動揺[どうよう] に 紙[かみ] と 筆[ふで] と を 震[ふる]わせて 如何[いか] に も 書[か]きにくそう に 書[か]いて いる。 汽車[きしゃ] は 幾[いく]マイル か 走[はし]る が、 筆[ふで] は 遅遅[ちち] と して 進[すす]まない。

その とき 一緒[いっしょ] に 乗[の]って いた 西洋人[せいよう-じん] が ふと、 かばん の 中[なか] から タイプライター を 取[と]り出[だ]して パチパチ と 打[う]ち出[だ]した。 その 早[はや]い こと 見[み]る間[ま] に 綺麗[きれい] な ハッキリ した ２ページ の 書物[しょもつ] と 控[ひか]え が 出来上[できあ]がった。

同[おな]じ 車室内[しゃしつ-ない] に 起[お]こった この ２つ の 出来事[できごと] の 差[さ] の あまり に 大[だい] で、 とても 同[おな]じ 時代[じだい] に 存在[そんざい] し得[え]る こと と は 思[おも]いなかった と いう こと で ある。
