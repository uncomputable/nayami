『 発刊[はっかん] に 当[あ]たりて 』（ 山下芳太郎[やました・よしたろう]  ）

我[わ]が 国[くに] は 維新以来[いしん-いらい]、 国運隆々[こくうん-りゅうりゅう] と して 進[すす]み、 幸[さいわ]い に も 今[いま] や 世界[せかい] 文明[ぶんめい] の 先進国[せんしん-こく] の 中[なか] に 加[くわ]わり 得[え]ました が、 これ だけ で は 満足[まんぞく] して いる こと は できませぬ。 この 上[うえ] さら に 位置[いち] を 進[すす]めて いよいよ 世界[せかい] 文明[ぶんめい] の 先頭[せんとう] に 立[た]って 諸国[しょこく] を 指導[しどう] する ほど の 大抱負[だい-ほうふ] が 必要[ひつよう] で あります。

しかし これ は 実現[じつげん] せしめる に 絶大[ぜつだい] の 努力[どりょく] を 要[よう]します こと は あたかも アルプス の 絶頂[ぜっちょう] に よじ登[のぼ]る が ごとき もの で、 心[こころ] や 身体[しんたい] を 練[ね]り固[かた]むる のみならず、 なるべく 装[よそお]い を 軽[かる]く して 進[すす]まねば なりませぬ。 しかるに 我[わ]が 国民[こくみん] は この 文化[ぶんか] の 峰[みね] を よじ登[のぼ]る に 実[じつ] に 西洋諸国[せいよう-しょこく] の 人々[ひとびと] の 知[し]らない 重荷[おもに] を 一[ひと]つ 余分[よぶん] に 背負[せお]って います。 それ は 難渋[なんじゅう] な 国字[こくじ] で あります。

外国[がいこく] で は わずか に ２６字[じ] で 用[よう] を 足[た]して いる の に 我[わ]が 日本人[にっぽんじん] は 少々[しょうしょう] も 数千[すうせん] の むつかしい 漢字[かんじ] を 覚[おぼ]え、 かつ それ を 使[つか]わねば なりませぬ。 これ が ため 日本[にっぽん] の 子供[こども] は 外国[がいこく] の ４年[ねん] に 対[たい]して およそ ６年[ねん] の 学習[がくしゅう] を 要[よう]し、 社会[しゃかい] の 実務[じつむ] に おいて は、 外国[がいこく] の 一人[ひとり] に 対[たい]し 二三人[に・さん-にん] の 人手[ひとで] を 要[よう]し、 彼[かれ] の 一時間[いち-じかん] に なし得[え]る 仕事[しごと] を 我[われ] は 一日[いちにち] を 要[よう]する と いう 大[おお]なる 弱点[じゃくてん] を 持[も]って おる の で あります。 我々[われわれ] が 一日[いちにち] も 早[はや]く この 重荷[おもに] を 投[な]げ捨[す]てざる 限[かぎ]り、 我々[われわれ] の 子孫[しそん] は 到底[とうてい] 文化[ぶんか] の 競争[きょうそう] に おいて 外国[がいこく] に 勝[か]ち得[え]る 見込[みこ]み が ありませぬ。

しかるに ここ に 幸[さいわ]い な こと に は、 我[われ]ら の 祖先[そせん] は 一千数百年[いっせん-すうひゃく-ねん] を 費[つい]やして 磨[みが]き上[あ]げた 世界無比[せかい-むひ] の 立派[りっぱ] な 片仮名[カタカナ] の 文字[もじ] を 我[われ]ら に 遺[のこ]して 授[さず]けて おります。 これ こそ 実[じつ] に 我[われ]ら の 救[すく]い の 船[ふね] で、 我[われ]ら の 活路[かつろ] は これ を 外[ほか] に して は 他[ほか] に は ありえませぬ。

仮名[カナ] の 光[ひかり] は 見[み]やすい 道[みち] を 伝[つた]えて 将来[しょうらい] の 我[わ]が 国民[こくみん] の 肩[かた] より、 漢字[かんじ] の 重荷[おもに] を 除[のぞ]かん が ため に 生[う]まれた の で あります。

私[わたし] は その 発行[はっこう] に 当[あ]たり、 開闢以来[かいびゃく-いらい] 我[わ]が 国[くに] を 守[まも]り たまえる 神[かみ] の 御霊[み-たま] の 前[まえ] に 祈[いの]り つつ、 読者諸君[どくしゃ-しょくん] に まみえます。

ご-同感[どうかん] の 方[かた] の ご-援助[えんじょ] を 切[せつ] に 願[ねが]う 次第[しだい] で あります。
