『 仮名[カナ] の 光[ひかり] 』

仮名[カナ] の 光[ひかり]！ 何[なん] の ため に 生[う]まれ いでました か。

仮名[カナ] の 尊[たっと]い 光[ひかり] を 世[よ] に 示[しめ]さん が ため です。 漢字[かんじ] に くらんで しまって いる 国民[こくみん] の 目[め] を 仮名文字[カナモジ] に 向[む]けしめる ため です。

仮名[カナ] は 我[わ]が 国[くに] に 生[う]まれ、 我[わ]が 国民[こくみん] に 育[はぐく]まれた 文字[もじ] で あります。 我[われ]ら は 祖先[そせん] から 授[さず]けられた この 尊[たっと]い 文字[もじ] の ある こと を 忘[わす]れて は なりませぬ。 仮名文字[カナモジ] は 世界[せかい] に 誇[ほこ]る べき 立派[りっぱ] な もの です。 その 立派[りっぱ]さ は 外[そと] から 塗[ぬ]り立[た]てた 彩[いろど]り で ない、 内[うち] から 発[はっ]する 光[ひかり] で あります。 この 文字[もじ] を 時勢[じせい] に 応[おう]じて、 書体[しょたい] に 改良[かいりょう] を 加[くわ]え、 記載法[きさい-ほう] を 工夫[くふう] する ならば、 一層[いっそう] その 価値[かち] を 増[ま]す に 相違[そうい] ありません。 国民[こくみん] が この 点[てん] に 向[む]かって 超[こ]えず 目[め] を 注[そそ]ぐ ならば、 仮名[カナ] は 漢字[かんじ] に 虐[しいた]げられた 境[さかい] から よみがえって、 いよいよ 光[ひかり] を 増[ま]し、 我[わ]が 民族[みんぞく] に 発生[はっせい] した 優雅[ゆうが] な 大和言葉[やまとことば] を 輝[かがや]かせ、 社会実務[しゃかい-じつむ] の 能率[のうりつ] を 高[たか]め、 一層[いっそう] 我[わ]が 文化[ぶんか] を 進[すす]める こと で ありましょう。

仮名[カナ] の 光[ひかり]！ その 使命[しめい] は なかなか 重[おも]い、 どうか 各位[かくい] の ご-援助[えんじょ] を 仰[あお]ぎます。 ご-愛護[あいご] を 請[こ]います。 我[わ]が 文化[ぶんか] の ため に、 国運発展[こくうん-はってん] の ため に。
