『 仮名[カナ] の 話[はなし] （１）』

日本[にっぽん] の 片仮名[カタカナ] が こんにち の 形[かたち] に まで、 発達進歩[はったつ-しんぽ] して きた 経路[けいろ] を 考[かんが]えて みます と、 ローマ字[じ] が 今[いま] の 形[かたち] に なって きた 順序[じゅんじょ] と 全[まった]く 同一[どういつ] で ある と いう こと は、 頗[すこぶ]る 興味[きょうみ] の ある こと で あります。 東西[とうざい] 土地[とち] を 異[こと] に し、 歴史[れきし] を 異[こと] に して いる に 関[かかわ]らず、 同[おな]じ よう な 道順[みちじゅん] を 経[へ]て 変化[へんか] して きた と いう こと は、 実[じつ] に 不思議[ふしぎ] な こと で あります。

片仮名[カタカナ] は 漢字[かんじ] の 一部分[いち-ぶぶん] を 取[と]って できた 極[きわ]めて 簡単[かんたん] かつ 明瞭[めいりょう] な 音符文字[おんぷ-もじ] で あります。 そして その 漢字[かんじ] は 元[もと] を 遡[さかのぼ]って みる と、 絵[え] から 出[で]て きて おります。 しかるに ローマ字[じ] も その 元[もと] に 遡[さかのぼ]る と、 エジプト文字[もじ] と なります。 エジプト文字[もじ] も その 新[あたら]しい もの は 音符文字[おんぷ-もじ] と なって おります が、 更[さら] に その 起源[きげん] を 尋[たず]ねる と 絵文字[えもじ] で あります。 古代[こだい]エジプト に おいて 鳥[とり] と いう 文字[もじ] は 𓅬 で、 魚[さかな] と いう 文字[もじ] は 𓆟 で あります。

また 𓀒 は 倒[たお]す と いう 文字[もじ] で、 𓇲 は 天[てん] より 水[みず] の 落[お]ちる 様[さま] を 書[か]いた もの で、 雨[あめ] を 表[あらわ]す 文字[もじ] で あります。

即[すなわ]ち この 時代[じだい] に は 一字[いち-じ] が 一語[いち-ご] を 表[あらわ]して いた の で あります。 しかし これ だけ で は 十分[じゅうぶん] に 用[よう] が 足[た]せない のみならず、 絵[え] に 書[か]く こと の できない 言葉[ことば] も ある から、 意味[いみ] を 表[あらわ]さないで、 ただ 音[おと] だけ 表[あらわ]す 文字[もじ] が 必要[ひつよう] と なって きます。 そこ で この 絵文字[えもじ] の 全部[ぜんぶ] か、 一部[いちぶ] を 取[と]って ある 音[おと] の 文字[もじ] と しました。 例[たと]えば 我[わ]が 国[くに] で 漢字[かんじ] の 毛[け] の 一部[いちぶ] を 取[と]って モ と した とき は 毛髪[もうはつ] と いう 意味[いみ] は 全然[ぜんぜん] 消[き]え失[う]せて、 ただ モ と いう 発音[はつおん] のみ を 表[あらわ]す 文字[もじ] と なりました。

それ と 同様[どうよう] に エジプト文字[もじ] で は 𓅓 は ミミズク と いう 文字[もじ] で、 Mulag と 発音[はつおん] して いました から ム と いう 音[おと] を 表[あらわ]す ため に、 その 頭[あたま] の 形[かたち] だけ を 書[か]く よう に しました。 これ が 現今[げんこん] の M と いう 文字[もじ] が できた 起源[きげん] で あります。

また 水[みず] と いう 字[じ] は 𓈗 で、 これ は Na と 発音[はつおん] して いました が、 ヌ の 音[おと] を 表[あらわ]す ため に、 その 絵文字[えもじ] の 波[なみ] の 一部[いちぶ] を のみ 書[か]く こと に しました。 これ が 今[いま] の N の 字[じ] の 起源[きげん] で あります。
